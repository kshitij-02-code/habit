# habit-frontend

A Habit & Study Tracker frontend built with React, Vite, and Material UI.

## Tech Stack

- React 19
- Vite 7
- Material UI 7
- React Router DOM
- Axios
- Framer Motion
- Day.js

## Getting Started

### Prerequisites

- Node.js (v18+)

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
